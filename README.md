# Aplikasi Stok Barang Warung

## Deskripsi
Aplikasi berbasis PHP dan MySQL untuk mengelola stok barang pada warung.

## Fitur
- Login
- Registrasi
- Dashboard
- Data Barang
- Barang Masuk
- Barang Keluar
- Grafik Stok
- Logout

## Cara Menjalankan
1. Install XAMPP.
2. Aktifkan Apache dan MySQL.
3. Import database `stock.sql`.
4. Salin folder proyek ke `htdocs`.
5. Buka:
   http://localhost/stockbarang/login.php
